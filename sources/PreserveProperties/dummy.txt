WiX Toolset Pills 15mg - Preserve Properties
====================================================================================================

This file is part of the "Preserve Properties" pill.

The pill demonstrates how to preserve the initial values of the properties, provided at install time,
in order to be used, later, for repair, upgrade or uninstall.

For more details please chech the GitHub repository:
https://github.com/WiX-Toolset-Pills-15mg/Preserve-Properties
